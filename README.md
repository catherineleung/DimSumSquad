Install dependencies:
```
npm install
```

Install typescript:
```
npm install -g typescript
```

Navigate to:
```
/types/
```

Initialize Git repo:
```
git init
```

Retrieve header files:
```
git clone https://github.com/borisyankov/DefinitelyTyped
```
